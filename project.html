  PROJECT 1: MOVIE REVIEW

This is our group project. It is a web-based application. It has basic function (CRUD) of a movie review website
including admin that can create, update, delete a movie from the list, the visitor that can view movies and
the registered user that can view and add a review to the movie. For backend: PHP, AJAX,
for frontend:Bootstrap, jQuery and for database: Mysql



  PROJECT 2: MOVIE RESERVATION

This is our group project. It is hosted web-based application. The end-user can view all web pages and
can book a movie reservation. After reservation,the end-user will recieve an email confirming its reservation .
Admin can view, createedit and update all the movies. Admin can also access the reservation logs.
For backend: PHP, frontend: HTML, CSS and javascript and database: MYSQL
